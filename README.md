# pokemon-md-red-hacks
Some hacks for Pokemon Mystery Dungeon: Red Rescue Team (US)

### Cheats (Codebreaker)
- Unlock / lock friend areas
- P1/P2 Inmortality (Max hp and full belly)

### Scripts (mgba)
- Unlock / lock friend areas
- P1/P2 Inmortality (Max hp and full belly)
- Guaranteed recruitment

### Usage
- Load script in mgba: Tools/Scripting/File/Load script...
- Unload scripts in mgba: Tools/Scripting/File/Reset
- Load cheats on mgba: Tools/Cheats/Load
- Load cheats in visualboy: Cheats/Cheat List.../CodeBreaker...

### Notes
- The scripts depend on the mgba API. [Download mgba here](https://mgba.io/downloads.html).
- Slide Codebreaker codes will not work in mgba, but it will do in visualboy. [Download visualboy here](https://visualboyadvance.org/download/).
